# mbti王

MBTI 性格测试小程序 · 管理后台 · API 服务

## GitHub 仓库

- **地址**: https://github.com/fnvtk/MBTI_wang
- **克隆**: `git clone https://github.com/fnvtk/MBTI_wang.git`

## 日常同步

```bash
# 拉取最新
git pull origin main

# 提交并推送
git add .
git commit -m "描述"
git push origin main
```

## 项目结构

- `admin/` - Vue 管理后台
- `api/` - ThinkPHP API
- `miniprogram/` - 微信小程序
