## 四、风险与保留

- Overview / Users / Enterprises 等页体量较大，本轮未直接修改其内部 scoped 样式，避免局部样式战。**token 已对齐**，即便不改，新旧页面并存也不会出现主色冲突。
- `SaTabs` 使用 Vue 3.4 的 `<script setup generic>`，`admin` 已依赖 `vue@^3.4.21`，兼容。
- 设计 token 为 CSS 变量，若后续品牌色调整（例如主色换真紫或品牌红），只需改 `admin-theme.css` 里的 `--sa-primary` 一处。
