## 一、小程序：按流程图「按步骤解锁」✅

### 1.1 结果页底部「步骤化三按钮」
新增工具：`miniprogram/utils/resultJourneyState.js`
- Step 1 · 看全文（资料完善 + 免费 / 已付费 → 自动解锁）
- Step 2 · 分享朋友圈（Step 1 完成后启用）
- Step 3 · AI 拍照（Step 2 完成后启用）
- 已完成步骤本地 storage 持久化 `mbti_journey_unlocks`

共享样式：`miniprogram/styles/result-page-dashboard.wxss` 新增：
- `.journey-stepper`（圆点 + 连接线，步骤条）
- `.result-tool-btn--locked`（未解锁态：灰底 + 🔒 emoji）

四个结果页（MBTI / DISC / PDP / SBTI）均接入：
- `data.journey = { step1Unlocked, step2Unlocked, step3Unlocked, activeStep }`
- `_syncJourney()` 在 applyResult / applyDetailPayload / initPayInfoFromRuntime / onShow / 支付成功时调用
- `onTapShareMoment` 点击时判断 step1；未解锁 toast 并触发 `onTapReadFull`
- `onTapFaceCamera` 点击时判断 step2；未解锁 toast
- `markShared / markCamera` 推进 step

**视觉**：步骤条 + 按钮 sub 文案（`完善资料即解锁` / `先解锁全文` / `先分享朋友圈` / `已解锁` 等）
