1. Hero 渐变卡：色系按测评区分（MBTI 紫 / DISC 蓝 / PDP 橙 / SBTI 青 / 面相洋红）+ 测评代码大字 + 称号 + 品类徽章 + 关键词 badge

7. 深度解读 CTA：紫蓝渐变卡，「深度解读方案 / 推广中心」两按钮

**各测评专属补充**：

**验收点**：
- MBTI 页面打开时视觉和管理端 Dashboard 同风格：能看到 Hero + 洞察四宫格 + 四维条 + 两栏 + 职业网格 + 引用 + CTA，整体紫色主调
